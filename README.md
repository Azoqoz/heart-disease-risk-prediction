# ❤️ Heart Disease Risk Prediction

An AI-powered machine learning web application for predicting heart disease risk based on patient health information.

## 🚀 Project Overview

This project uses machine learning techniques to estimate the probability of heart disease risk using patient clinical data.

The main objective of this project was not only achieving high accuracy, but also minimizing False Negatives and improving Recall, which is especially important in medical prediction systems.

## 🧠 Features

- Interactive web application using Streamlit
- Heart disease risk prediction
- Probability-based prediction results
- Model performance visualization
- Clean and modern UI
- Multiple ML model comparison and evaluation
- Recall-focused optimization strategy

## ⚙️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- NumPy
- Joblib

## 📊 Machine Learning Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Data Scaling
5. SMOTE for class balancing
6. Model Comparison
7. Hyperparameter Tuning
8. Model Evaluation
9. Deployment

## 🎯 Final Model

The final deployed model uses:

- XGBoost Classifier

Optimized to reduce False Negatives and improve Recall performance.

## 📈 Model Performance

- Accuracy: 98.54%
- Precision: 100%
- Recall: 97.09%
- F1 Score: 98.52%
